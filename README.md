Простой блог на основе Spring Boot, созданный в рамках обучения на Hexlet .
Приложение демонстрирует основы веб-разработки на Java с использованием Spring MVC, Spring Data JPA и Thymeleaf.


Spring Boot — основной фреймворк
Spring MVC — для обработки HTTP-запросов
Spring Data JPA — для работы с базой данных
Thymeleaf — шаблонизатор для отображения страниц
H2 Database — встроенная БД для разработки (возможно, PostgreSQL в продакшене)
Lombok — для сокращения boilerplate-кода
